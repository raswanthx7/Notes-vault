# 📝 Markdown Format Guide

Use this as a reference for writing clean, professional notes in `.md` files.

---

## 📌 Headings

Use `#` symbols (1 to 6) for headings:

```md
# H1 Heading
## H2 Heading
### H3 Heading
#### H4 Heading
```

## Text Formatting

```md
**Bold Text**       - Bold  
*Italic Text*       - Italic  
~~Strikethrough~~   - Strikethrough  
`Inline Code`       - Inline Code (like commands)
```

## List

### Unordered List

```md
- Item 1
- Item 2
  - Sub-item
* Item 3
+ Item 4
```

### Ordered List

```md
1. First step
2. Second step
   1. Sub-step
   2. Sub-step
```

## Links

```md
[Google](https://www.google.com)
```

## Images

```md
![Alt Text](https://link-to-image.com/image.png)
```

## Horizontal Line

```md
---
```

## Blockquotes

```md
> This is a quote or highlight.
```

## Task List

```md
- [x] Completed task
- [ ] Incomplete task
```
